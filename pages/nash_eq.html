Tips

  • Define the number of strategies of each player in corresponding input boxes.
  • Create a new table for input players' outcomes by clicking the Create/Clean button.
  • Fill the gaps in the table with values in the format like "val1/val2". For separation, "/" or ";" can be used. "Populate strategies" button will fill the table with random values automatically.
  • OR use ready presets that have filled tables with different outcomes.
  • Press the RUN button to start visualization.
  • The process of iteration can be controlled via the three buttons below the dynamic table or with the slider bar.
