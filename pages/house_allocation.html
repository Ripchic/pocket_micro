Tips

  • Define the number of agents by adding or removing them.
  • Press the Populate endowment/preferences to fill the corresponding table field.
  • Press the RUN button to start the algorithm.
  • Initially pairs Player:House are displayed in a cycle shape.
  • In blue are marked arrows pointing top available houses.
  • Red arrows denotes the cycle formation. Red cells in table show that house is not awailable more.
  • Final allocation (houses in graph and table) is colored in green.
