Tips

  • Define the number of agents and items.
  • Press the Populate Random button to fill the table with random values or fill it yourself.
  • You can change the order of players by dragging rows in th table.
  • Press the RUN button to start the algorithm.
  • In grey is marked cell that we currently observe.
  • Red cells are declined as these items are already allocated with other players.
  • Final allocation is colored in green.
